* CRUD
	- CREATE, READ, UPDATE, DELETE
	
* CREATE
	- INSERT INTO 테이블 (컬럼명들) VALUES (값들)
	
* READ
	- SELECT 컬럼 FROM 테이블명
	
	- 옵션
		> WHERE로 조건을 줌
		> AND로 조건 추가
		> OR로 조건 추가
		
		> ORDER BY DESC/ASC
		> LIMIT: 조회할 개수 제한
		> OFFSET: 앞의 로우들 스킵 가능
	
* UPDATE
	- UPDATE 테이블명 SET 컬럼=새값 WHERE 조건
	
* DELETE
	- DELETE FROM 테이블명 WHERE 조건
