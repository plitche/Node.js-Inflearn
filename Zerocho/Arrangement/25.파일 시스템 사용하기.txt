* fs
	- 파일 시스템에 접근하는 모듈
	- 파일/폴더 생성, 삭제, 읽기, 쓰기 가능
	- 웹 브라우저에는 제한적이었으나 노드는 권한을 가지고 있음
	- 파일 읽기 예제
	
	const fs = require('fs')

	fs.readFile('./readme.txt', (err, data) => {
		if (err) {
			throw err;
		}

		console.log(data)
		console.log(data.toString());
	})
	

* 동기 메서드와 비동기 메서드
	- 이전 예제를 여러번 실행해보기
		> 매 번 순서가 다르게 실행됨
		> 순서에 맞게 실행하려면 ?
		
	- 동기와 비동기: 백그라운드 작업 완료 확인 여부
	- 블로킹과 논 블로킹: 함수가 바로 return 되는지 여부
	- 노드에서는 대부분 동기-블로킹 방식, 비동기-논블로킹 방식임
